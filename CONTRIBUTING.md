# Contributing

Welcome! The goal of this repository is to build the **most comprehensive and rigorous GitHub Actions AI Skill**.

## Ways to Contribute

| Method | Description |
|--------|-------------|
| **Submit Issue** | Report bugs, suggest new templates, request new language support |
| **Submit PR** | Add templates, optimize decision tree, supplement FAQ, fix errors |
| **Translation** | Translate `PROMPT.md` into other languages |
| **Examples** | Submit real-world workflow examples to `examples/` |

## Skill Development Standards

Modifications to `SKILL.md` and `PROMPT.md` must follow the **12 writing principles**:

1. Decision-tree driven (new scenario -> new branch)
2. Imperative instructions
3. Before/After comparison
4. Few-Shot coverage (common + edge cases)
5. Checkpoint verification
6. Security red lines unbroken

## PR Process

1. Fork the repository
2. Create a branch (`feat/xxx` or `fix/xxx`)
3. Self-check against the [Verification Checklist](SKILL.md#10-verification-checklist) after changes
4. Submit PR with a description of changes
