# 贡献指南 / Contributing

欢迎贡献！本仓库的目标是打造**最全面、最严谨的 GitHub Actions AI Skill**。

## 贡献方式

| 方式 | 说明 |
|------|------|
| 🐛 **提交 Issue** | 报告 Bug、建议新模板、请求新语言支持 |
| ✨ **提交 PR** | 新增模板、优化决策树、补充 FAQ、修复错误 |
| 🌍 **翻译** | 将 `PROMPT.md` 翻译到其他语言 |
| 📝 **示例** | 向 `examples/` 提交真实场景的工作流 |

## Skill 开发规范

提交到 `SKILL.md` 和 `PROMPT.md` 的修改，必须遵循 **12 项编写原则**：

1. 决策树驱动（新场景 → 新分支）
2. 祈使句指令
3. Before/After 对比
4. Few-Shot 覆盖（常见 + 边界）
5. 检查点验证
6. 安全红线不破

## PR 流程

1. Fork 仓库
2. 创建分支 (`feat/xxx` 或 `fix/xxx`)
3. 修改后对照 [验证清单](SKILL.md#10-验证清单) 自检
4. 提交 PR，附上变更说明
